## Como executar
1. Clone o repositório.
2. Abra a pasta do exercício ou projeto na sua IDE (IntelliJ, Eclipse, VS Code).
3. Localize a classe com método `main` e execute.

## Requisitos
- JDK instalado e configurado no ambiente.
